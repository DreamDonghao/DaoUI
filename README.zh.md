## 介绍

DaoUI 是一个由 C++ 开发的跨平台 C++ GUI库，你可以使用 DaoUI 快速开发桌面应用程序。

DaoUI 基于 SDL3 开发，可以使用硬件加速，并且全部内容都由顶点数组批量绘制，使用恰当有着不错的性能，因此你也可以用于游戏开发。但是由于 DaoUI 并不是一个专业的游戏引擎，因此在程序优化方面可能有更高的要求。你可用 DaoUI 制作您的 2D 游戏或其他可视化应用。

DaoUI 并非像 Qt 那样有着非常多的组件供开发者使用，而是只提供了纹理，几何图形，按钮等基础的内容。因此 DaoUI 有着极低的学习成本，只要你有 C++ 使用经验几乎可以极短时间内上手。并且各种常用组件正在开发中。

你可以大胆参与到 DaoUI 的开发中来，不论你是没有任何开发经验的学生还是对编程感兴趣，都可将本项目用于你增长开发经验。
